# Deep-JSCC video transmission modules on GNU Radio

C++ blocks:

* packet header formatter and parser

* single-tap soft equalizer

Python blocks:

* deep-jscc source

* deep-jscc sink



VAE models not included

## Prerequisite

* SDR driver

* [GNU Radio 3.8](https://jtaizhang.github.io/project/gnuinstall)
* onnx_tensorrt 7.1
